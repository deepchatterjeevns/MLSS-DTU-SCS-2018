#### [PyTorch](https://pytorch.org/) can be installed via:  
* OS Linux, Python 3.6, CUDA 8.0:  
  * *conda install pytorch torchvision cuda80 -c pytorch*  
* OS Linux, Python 3.6, CUDA 9.0:  
  * *conda install pytorch torchvision -c pytorch*
